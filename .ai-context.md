# RZGB AI Context File

## Stack
- Frontend: HTML/CSS/JS (Bootstrapmade template)
- Backend: Node.js on Hostinger
- Database: Supabase
- Deploy: GitHub push → Hostinger auto-deploy

## Project URLs
- Main site: rzglobalsolutions.co.uk
- Portal: portal.rzglobalsolutions.co.uk

## Business Context
- B2B supply chain intermediary
- Connects Indian manufacturers (Rajkot, Gujarat) with UK/EU/US clients
- Sectors: automotive, CNC machining, castings, hydraulics, industrial components
- Three user types: Client, RZ Global (admin), Supplier

## Portal Features
- Client login → submit orders → track workflow stages
- Order stages: Quote → Casting/Forging → Machining → QC → Packaging → Delivery
- Supplier portal → receive jobs, update progress
- File upload on quote form
- Tidio chatbot integrated on main site
